Download the data

This page contains all the data referenced within the Nature Communications publication "Proteome-wide and matrisome-specific alterations during human pancreas development and maturation", which is also referenced on the 'data' page of this web application. All data is available in .xslx format. Please contact the authors with questions and concerns.

Identified ECM

This dataset contains the full list of all identified ECM proteins. Proteins common to all groups (quantified) are highlighted.

Quantified ECM

This dataset contains the list of all quantified ECM proteins. Log2 DiLeu reporter ion intensities of each protein in all samples are shown. Significantly changed proteins among age groups are highlighted. It also contains information of all pairwise comparisons between different groups.
